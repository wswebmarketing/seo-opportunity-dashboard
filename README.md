# SEO Opportunity Dashboard

## Objetivo
Identificar oportunidades de palavras-chave com base na análise da SERP.

## Funcionalidades
- Análise de palavras-chave
- Score de oportunidade
- Análise de SERP
- Integração com SerpAPI

## Arquitetura
- services → lógica de negócio
- utils → parsing e integração
- pages → interface (Dash)

## Tecnologias
- Python
- Dash
- Dash Bootstrap Components
- Pandas
- SerpAPI
- Requests

## Como rodar
pip install -r requirements.txt
python app.py
